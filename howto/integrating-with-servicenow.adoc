= 与ServiceNow集成
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
OnCommand Insight 与ServiceNow管理软件相集成、可提供比单独使用的产品更高的价值。

通过使用Python脚本、Insight可以将数据与ServiceNow集成、从而同步以下信息：

* ServiceNow服务器的存储资产数据
* ServiceNow服务器的主机和虚拟机URL
* 主机/VM与存储之间的关系
