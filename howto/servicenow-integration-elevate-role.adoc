= 提升角色
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您必须先将ServiceNow角色提升为security_admin、然后才能与Insight集成。



== 步骤

. 使用管理员权限登录到您的ServiceNow实例。
. 在*系统管理员*下拉列表中、选择*提升角色*并将您的角色提升为security_admin。单击确定。
