= 从剪贴板打开报告（ xml ）
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以打开先前复制到剪贴板的报告规范。



== 关于此任务

您可以通过创建新报告或打开现有报告来进入报告用户界面



== 步骤

. 打开Cognos 11报告门户： https://server-name:9300/bi/[]
. 在工具栏中、单击 image:../media/new-report.gif[""]
. 单击 * 报告 *
. 单击 * 页面 * 图标image:../media/pages-icon.gif[""]
+
*报告*图标 image:../media/report-icon.gif[""] 将显示

. 左键单击*报告*图标
+
此时将显示报告选项。

. 单击 * 从剪贴板打开报告 * 。
