= 查看数据仓库数据库架构文档
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
您可以查看数据仓库数据库架构信息。



== 步骤

. 登录到数据仓库门户、网址为 `+https://hostname/dwh+`、其中 `hostname` 是安装了OnCommand Insight 数据仓库的系统的名称。
. 在数据仓库工具栏上、单击 image:../media/oci-7-help-icon-gif.gif[""] 并选择*架构*。
