= 备份OnCommand Insight 数据库
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
备份Insight数据库、以确保在升级后发生问题描述 时您拥有最新的备份。在备份和还原阶段、不会收集性能数据；因此、备份应尽可能接近升级时间。



== 步骤

. 在浏览器中打开Insight。
. 单击*管理*>*故障排除*。
. 在*故障排除*页面上、单击*备份*。
+
备份数据库的时间可能因可用资源(架构、CPU和内存)、数据库大小以及环境中监控的对象数量而异。

+
备份完成后、系统会询问您是否要下载此文件。

. 下载备份文件。
