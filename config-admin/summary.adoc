= 摘要
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
资产页面上的"摘要"部分显示有关特定资产的信息摘要、并显示与资产相关的问题(用红色圆圈表示)、其中包含指向追加信息 的相关资产以及分配给资产的任何性能策略的超链接。

以下示例显示了虚拟机资产页面的摘要部分中提供的一些类型的信息。旁边带有红色实心圆圈的任何项目均表示受监控环境存在潜在问题。
