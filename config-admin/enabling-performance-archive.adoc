= 启用性能归档
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要启用性能数据归档、请执行以下步骤。



== 步骤

. 在工具栏上、单击*管理*>*设置*。
. 选择*备份和归档*选项卡。
. 在性能归档部分中、确保选中了"*启用性能归档"。
. 指定有效的归档位置。
+
您不能在Insight安装文件夹下指定文件夹。

+
最佳实践：不要为归档指定与Insight备份位置相同的文件夹。

. 单击 * 保存 * 。
+
归档过程在后台处理、不会干扰其他Insight活动。
