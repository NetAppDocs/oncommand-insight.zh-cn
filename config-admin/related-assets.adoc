= 相关资产
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果适用、资产页面将显示相关资产部分。例如、卷资产页面可能会显示有关存储池、已连接交换机端口和计算资源等资产的信息。每个部分都包含一个表、其中列出了该类别中的任何相关资产、指向相应资产页面的链接以及与该资产相关的多个性能统计信息。
