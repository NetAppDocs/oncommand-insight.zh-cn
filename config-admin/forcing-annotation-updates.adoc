= 正在强制更新标注
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
如果您更改了标注并希望立即在报告中使用它们、请使用强制标注选项之一。



== 步骤

. 在Web UI中、单击*管理*并选择*故障排除*。
. 在页面底部、单击*高级故障排除*链接。
. 单击*操作*选项卡。
. 选择以下选项之一：
+
** *更新DWH标注*以强制更新数据仓库中用于报告的标注。
** *更新DWH标注(包括Deleted)*以强制更新数据仓库中的标注(包括已删除的对象)以用于报告。
