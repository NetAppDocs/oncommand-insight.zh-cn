= 计算数据库对象的数量
:allow-uri-read: 
:icons: font
:imagesdir: ../media/


[role="lead"]
要确定OnCommand Insight 数据库中的对象数量、请使用计算比例功能。



== 步骤

. 在Insight工具栏上、单击*管理*并选择*故障排除*。
. 在页面底部的"Other tasks"区域中、单击*高级故障排除*链接。
. 在高级故障排除页面上、单击*报告*选项卡。
. 单击*计算的比例*。
